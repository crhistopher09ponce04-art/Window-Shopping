## Despliegue en Render
1. Subir este repo a GitHub.
2. Conectar a Render → New → Blueprint.
3. Render detectará `render.yaml` automáticamente.
4. Agregar en Environment la variable:
   - SECRET_KEY = clave_super_segura_123456
